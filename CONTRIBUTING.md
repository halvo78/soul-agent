# Contributing to soul-agent

## How to Contribute

### Share Your Soul Files
The most valuable contribution: share how you use soul-agent.

1. Create a directory in `examples/` with your use case
2. Include your SOUL.md (anonymize if needed)
3. Describe what worked and what didn't
4. Open a PR

### Add Templates
New templates for specific use cases are always welcome:
- Customer support agent
- Code review agent
- Content writer agent
- Data analyst agent
- DevOps agent

### Improve Documentation
- Fix typos, improve clarity
- Add integration examples for new frameworks
- Translate to other languages

### Report Issues
- What did you try?
- What did you expect?
- What happened instead?

## Code of Conduct
Be kind. Be helpful. We're all figuring this out together.
